Optimisation d'Itinéraire (Tri Topologique)
Ce projet Python illustre la planification de tâches avec dépendances en utilisant le tri topologique. L’objectif est de déterminer un ordre d’exécution valide pour toutes les tâches, même lorsque certaines dépendent d’autres.


Fonctionnalités
Représentation des tâches et de leurs dépendances sous forme de graphe (liste d’adjacence).
Utilisation d’un parcours en profondeur (DFS) récursif pour explorer le graphe.
Calcul d’un ordre d’exécution correct qui respecte toutes les dépendances.
Possibilité de calculer le temps total d’exécution en fonction de la durée de chaque tâche.
Affichage clair de l’ordre final et des temps cumulés pour chaque tâche.


Exécution
Pour lancer le programme : <br/>
python3 sujet5.py
